1. Deskripsi Aplikasi<br>
<br>
Animeverse adalah aplikasi katalog anime yang dapat digunakan untuk menjelajahi dan mengelola daftar anime favorit. Proyek ini dibangun menggunakan Flutter untuk sisi front-end, serta Firebase Authentication dan Cloud Firestore untuk proses autentikasi dan penyimpanan data pengguna.<br>
